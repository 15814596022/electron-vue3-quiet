# desk-app

> 这是一个基于 electron 的 vue3 的模板，使用当下最新的技术栈.

## 技术栈关键字

- typesctript
- vite + rollup
- electron + electron-builder
- vue3 + setup + tsx
- windicss
- prettier
- eslint
- sqlite3
- ．．．

> ### **请确保您的 node 环境是大于或等于 16**

## 安装使用

- 获取项目代码

```bash
git clone https://gitee.com/TaiAi/desk-app.git
```

- 安装依赖

```bash
cd desk-app

npm i -g @antfu/ni

ni
```

- 运行(开发)

```bash
nr dev
```

- 打包(生产)

```bash
nr build
```
